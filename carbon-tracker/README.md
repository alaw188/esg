# 地產碳排放管理平台 · Scope 1 & 2

面向房地產開發商的溫室氣體排放管理工具，依 **GHG Protocol Corporate Standard** 計算 **範疇一（直接排放）** 與 **範疇二（外購能源間接排放）**。

## 功能概覽

| 模組 | 說明 |
|------|------|
| 總覽儀表板 | 年度/項目篩選、範疇分佈、月度趨勢、排放強度 |
| 項目管理 | 住宅、商業、地盤施工、總部等開發項目 |
| 範疇一 | 天然氣、柴油、汽油、LPG、冷媒逸散等 |
| 範疇二 | 外購電力（位置/市場基礎）、蒸汽、暖通、供冷 |
| 報告匯出 | CSV 明細報告、JSON 完整備份 |
| 係數設定 | 香港/大灣區/台灣/新加坡電網係數，可自訂燃料係數 |

## 快速開始

### 方法一：直接開啟

雙擊 `index.html` 在瀏覽器中開啟（建議 Chrome / Edge）。

### 方法二：本地伺服器

```powershell
cd d:\001\carbon-tracker
python -m http.server 8080
```

瀏覽器前往 `http://localhost:8080`

## 範疇定義（房地產場景）

### 範疇一 · 直接排放

- **固定燃燒**：地盤發電機、辦公室鍋爐、示範單位暖通
- **移動燃燒**：公司車隊、地盤運輸車輛
- **逸散排放**：冷媒補充、滅火劑

### 範疇二 · 間接排放

- **外購電力**：地盤、銷售中心、寫字樓用電
- **外購能源**：蒸汽、區域供冷供熱

> 本工具**不包含範疇三**（建材運輸、租戶排放、投資物業等價值鏈排放）。如需 Scope 3，可後續擴展。

## 排放係數（按年度獨立）

每個報告年度擁有**獨立的排放係數設定**，修改某一年不會影響其他年度：

| 行為 | 說明 |
|------|------|
| 新年度 | 自動沿用**上一年**的係數作為起點 |
| 跟隨上一年 | 一鍵從上一年複製（覆蓋當前編輯中的數值） |
| 儲存本年度設定 | 固定該年度的係數，供該年所有排放記錄計算使用 |
| 還原本年度 | 只還原選定年度為系統預設，不影響其他年 |

排放記錄會依**該筆記錄所屬年度**的係數計算，確保歷年報告數字不因係數更新而改變。

## 預設排放係數（系統預設值）

| 排放源 | 係數 | 單位 |
|--------|------|------|
| 香港電網（位置基礎） | 0.66 | kg CO₂e/kWh |
| 天然氣 | 2.034 | kg CO₂e/m³ |
| 柴油 | 2.687 | kg CO₂e/升 |
| 汽油 | 2.296 | kg CO₂e/升 |
| LPG | 1.512 | kg CO₂e/kg |

係數可在「排放係數設定」頁面調整，以配合貴司 ESG 報告或第三方核證要求。

## 數據儲存

所有數據儲存於瀏覽器 **localStorage**，不會上傳至任何伺服器。建議定期使用「匯出 JSON 備份」。
